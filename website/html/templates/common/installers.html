Download installer for Mac

Instructions:
(Note: You must have Python 2.5, 2.6, or 2.7 installed)

  1. Download the {{ TESTBED }} tarball to your computer
  2. Extract the tarball to the directory in which you would like to install {{ TESTBED }}.
  3. Navigate to that directory and run ./install.sh (or python seattleinstaller.py).

To check that {{ TESTBED }} is running, try running:
ps Aww | grep nmmain.py | grep -v grep
Download installer for Windows

Instructions:

  1. Download the {{ TESTBED }} zip file to your computer
  2. Extract the zip file to the directory in which you would like to install {{ TESTBED }}.
  3. Double click (or run from the command line) install.bat.

To check that {{ TESTBED }} is running, look in Task Manager for the process pythonw.exe.
